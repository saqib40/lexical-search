## To keep it simple, building the inverted-index (or during lookup while searching) follows the below steps in order;
- Tokenization
- Normalization
- Stop Word Removal
- Stemming and Lemmatization
- Put it in memory (or on disk) if you building (or lookup and evaluate the boolean expression if you retrieving)
